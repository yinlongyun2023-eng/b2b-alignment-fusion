# b2b-alignment-fusion

B2B 合作理念对齐与战略融合 Skill。从多份文档（录音转写、方法论、会议记录）中提取合作双方的核心理念，生成使命·愿景·价值观同频对照表和战略框架文档。

## 功能

- 整理多份录音/文档为融合版
- 梳理双方理念同频点
- 生成 B2B 合作战略框架
- 建立使命愿景价值观对照表

## 触发词

理念对齐、同频对照、战略融合、融合版、意图梳理、B2B合作对齐、双方理念融合、使命愿景价值观对照

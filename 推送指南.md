## 你需要做的（Step 3）

### 1. 打开 GitHub 创建新仓库

1. 打开浏览器，访问：**https://github.com/new**
2. 登录你的 GitHub 账号（用户名：yinlongyun2023-eng）
3. 填写：
   - **Repository name**：`b2b-alignment-fusion`（建议用这个，和项目名一致）
   - **Description**：B2B合作理念对齐与战略融合 Skill（可选）
   - **Public**（公开）
   - ⚠️ **不要勾选** "Add a README file"（我们本地已有）
4. 点击 **Create repository**

### 2. 记下仓库地址

创建成功后，GitHub 会显示一个地址，类似：

```
https://github.com/yinlongyun2023-eng/b2b-alignment-fusion.git
```

**复制这个地址**，后面要用。

## 最后一步（Step 4）

在 Cursor 终端中执行（把下面的地址换成你实际的）：

```powershell
cd "D:\b2b-alignment-fusion"
git remote add origin https://github.com/你的用户名/b2b-alignment-fusion.git
git branch -M main
git push -u origin main
```

例如，如果你的用户名是 `yinlongyun2023-eng`，则：

```powershell
cd "D:\b2b-alignment-fusion"
git remote add origin https://github.com/yinlongyun2023-eng/b2b-alignment-fusion.git
git branch -M main
git push -u origin main
```

- 首次 `git push` 可能会弹出登录窗口，按提示登录 GitHub 即可
- 成功后，刷新 GitHub 网页就能看到你的代码了 🎉

## 以后更新怎么推送？

```powershell

git commit -m "说明你改了什么"
```
